游戏简介

（游戏演示见snake.mp4）
操纵贪吃蛇吃到更多零食吧！但顽皮的森林女神，给贪吃蛇设置了障碍。
每当它要吃到零食的时候，森林女神就会考它一道算术题，只有在规定时间内答对，才能继续吃零食。
聪明的你，快来帮贪吃蛇闯过难关吧！



操作说明

上下左右或WASD控制贪吃蛇转弯。
吃到白色的零食的时候，会出现算术题，输入计算的答案并回车，玩家得分，游戏继续。
如果贪吃蛇撞到墙壁，或者头撞到自己的身体，游戏结束。
如果没有在规定时间内回答算术题，或者回答错误，游戏结束。

首先需要使用pip install pygame，安装依赖的库。

在quiz.py中我刻意删除了两段代码。代码中的“# TODO”就是我删除代码的位置。请尝试复原这两段代码。
